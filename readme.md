# stehlik_web_project
 Pro správné fungování webu je potřeba import databáze ze složky "dev".
 Admin účet ( pro přístup do admineru a pro admin práva ) -> admin:Restrikce2021
 Tohle je můj prezentacni projekt. Jelikož řešíme práci PHP ( nette ) programátory, tak jsem skoro vůbec neřešil vizuální stránku ( i když, někdy do budoucna bych chtěl ten web udělat pěkný ).
 Web jako takový má ukázat mou schopnost práce s PHP a Nette. Proto ten web tvoří funkce, které spolu úplně nesouvisí.
 Bohužel kvůli určitým komplikacím, jsem nepřidal na web vše co jsem chtěl přidat. Jako například funkční mailer, kde by uživatel mohl poslat email na mou adresu, počítání komentářů u příspěvku, potvrzovací email pro nové uživatele ( hned po registraci by se jim dal stat neověřeného uživatele, který nemá žádná práva a po nějaké době se sám zruší ) a další drobné funkce.
